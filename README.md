# Learning

A collection of programming projects spanning my learning journey — from writing my first lines of Java and C++ through implementing classic graph algorithms and teaching others. It ranges across language fundamentals, data-structure and graph-algorithm implementations, small utilities, and material I prepared as a high-school programming instructor.

## Java

### `Java-Basics/`
Foundational Java projects built through a structured course, including GUI programs, sound playback, and user input handling. Entry point into object-oriented programming.

### `Learn2022/`
Java practice workspace from 2022, used for experimenting with new concepts alongside formal coursework.

### `17-Projects/`
Selected projects from a guided Java curriculum — includes a **Length Converter**, **Word Counter**, and an image display program. Focused on practical application of core Java APIs.

### `Algorithms/`
Implementations of classic graph and tree algorithms in Java:

| Algorithm | Category |
|---|---|
| Binary Search Tree (BST) | Data Structures |
| Dijkstra's Shortest Path | Graph — Weighted |
| Prim's Minimum Spanning Tree | Graph — MST |
| Kruskal's + Disjoint Set Union | Graph — MST |
| Tarjan's Lowlink / SCC | Graph — Strongly Connected |
| Kosaraju's SCC | Graph — Strongly Connected |
| Eulerian Paths | Graph — Traversal |
| Directed & Undirected Cycle Detection | Graph — Validation |

### `Highschool-Programming-Teaching/`
Material prepared and taught as an instructor in a **high school programming club**. Includes introductory exercises and guided problems written to help beginners get started with Java.

### `Hobbies/`
Small utilities written for personal use — prime number tools and helper scripts for everyday tasks.

### `Personal-Interest/`
Math-focused programs written out of curiosity:
- **Stars and Bars** — combinatorics calculator
- **Galois 2022 #4** — solution to a Galois Contest problem
- **Change Finder** — coin change problem variant
